# any-website-workflow — Claude Code skill

Design and BUILD a distinctive, up-to-date best-practice marketing website for any business —
from scratch or transformed from an existing site — then deploy it live to Cloudflare Pages
(free, HTTPS). Static, self-contained, mobile-aware, GDPR-aware (self-hosted fonts by default).
